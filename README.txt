AMERICA 250 FOOD & WELLNESS TRACKER

This is a standalone Progressive Web App (PWA). It does not require Excel or Power Apps.

WHAT IT DOES
- Daily food, calorie, protein, carbs, fat and fiber tracking
- Water and weight tracking
- UPC/EAN barcode lookup through Open Food Facts
- Camera scanning where the mobile browser supports BarcodeDetector
- Manual barcode entry fallback
- Shopping list with estimated and actual costs
- Supplement tracker
- CSV food-log export and JSON backup/restore
- Data stored locally on the device

HOW TO PUBLISH
The app must be served over HTTPS for camera access and reliable installation.
Easy choices: GitHub Pages, Netlify, Cloudflare Pages, or another static website host.
Upload every file in this folder without changing the names.

ANDROID INSTALL
1. Open the HTTPS app address in Chrome or Edge.
2. Open the browser menu.
3. Tap Install app or Add to Home screen.

IPHONE INSTALL
1. Open the HTTPS app address in Safari.
2. Tap Share.
3. Tap Add to Home Screen.

IMPORTANT
- Barcode product information comes from Open Food Facts and may be incomplete. Verify with the package label.
- Camera barcode detection varies by browser. Manual UPC entry always remains available.
- Data stays only on the device unless exported. Make regular JSON backups.
